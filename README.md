# 有道词典笔 - 系统镜像资源

本仓库用于存放有道词典笔相关的系统固件资源，包括根文件系统（rootfs）和启动镜像（boot）。

## 资源下载

所有镜像文件均发布在 **Releases** 页面，请前往以下地址下载：

👉 [Releases 页面](https://github.com/你的用户名/你的仓库名/releases)

## 文件说明

| 文件类型 | 说明 | 注意事项 |
|---------|------|----------|
| rootfs | 根文件系统镜像 | **可能未改写读写权限** |
| boot | 启动分区镜像 | 包含内核及启动相关文件 |

> ⚠️ **重要提醒**：请务必根据 Release 标题和说明中的标注选择对应版本，不同机型/固件版本可能存在差异。

## 使用前须知

1. 确保设备型号与 Release 标注的适用机型一致
2. 刷写前请备份当前系统
3. 未改写权限的 rootfs 需要配合正确的权限处理方式使用
4. 刷写操作有一定风险，请自行承担

## 版本标注格式

Release 版本号将按以下格式标注：

- `设备型号_权限_rootfs` — 根文件镜像
- `设备型号_权限_boot` — 启动镜像
